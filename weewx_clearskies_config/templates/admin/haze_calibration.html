Haze Calibration

The haze detection system uses particulate matter (PM2.5) data and a hygroscopic scattering model to identify haze conditions. Calibration collects clear-sky solar transmittance samples over time to establish a local baseline.

Haze detection Default: enabled. When disabled, haze conditions will not be reported regardless of PM2.5 levels.
Calibration parameters

These settings control how the system collects and uses clear-sky transmittance samples to build a local baseline.

Default: {{ defaults.calibration_percentile }}. The percentile of clear-sky samples used to set the baseline Kcs. Higher values select only the clearest days. Default: {{ defaults.calibration_window_days }} days. Samples older than this window are discarded from calibration. Shorter windows adapt faster to sensor drift; longer windows are more stable. Default: {{ defaults.calibration_min_samples }}. Haze detection is suppressed until this many samples have been collected. Too few samples can produce unreliable baselines.
Hygroscopic correction Default: {{ defaults.gamma }}. Hygroscopic correction exponent (Hänel 1976). Default 0.45 is suitable for most locations. Lower values reduce humidity sensitivity; higher values increase it.
