Sky Classification Calibration

The sky cover classifier uses the Kasten-Czeplak clearness index (Km) — the ratio of measured solar irradiance to the clear-sky theoretical maximum. Adjust the thresholds below to match your sensor's characteristics.

{# Sensor accuracy note #}
Sensor accuracy note: Davis VP2 pyranometers: ±3–5%. Ambient Weather stations: approximately ±15%. If you use consumer-grade equipment, tight threshold values may produce unreliable classifications during marginal sky conditions. Consider relaxing the thresholds by 0.05–0.10 for less precise sensors.

SCATTER_CLOUDS Km sub-split thresholds

These thresholds divide the Km range for partly-cloudy conditions into FEW, SCT, and BKN sub-categories. Relationship must hold: scatter_bkn_max < scatter_sct_max < scatter_few_max.

Default: {{ defaults.scatter_few_max }}. Above this Km value, sky is classified as FEW or CLR. Default: {{ defaults.scatter_sct_max }}. Between this and scatter_few_max, sky is classified as SCT. Default: {{ defaults.scatter_bkn_max }}. Between this and scatter_sct_max, sky is classified as BKN. Below this, sky enters the OVERCAST thresholds.
OVERCAST sub-split thresholds

Below scatter_bkn_max, the classifier checks both Km and Kv (variability index) to distinguish broken-to-overcast from fully overcast sky.

Default: {{ defaults.overcast_km_threshold }}. Km below this value is candidate for OVC classification. Default: {{ defaults.overcast_kv_threshold }}. Kv below this value (combined with low Km) confirms OVC. Higher Kv at low Km suggests broken cloud rather than solid overcast.
Solar zenith angle guard Default: {{ defaults.sza_min_elevation }}°. Below this sun elevation the classifier skips the Km method (sunrise/sunset atmospheric path length distorts the index). Typical range: 3°–10°.
