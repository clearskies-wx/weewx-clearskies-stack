TLS Settings

Configure HTTPS for the configuration UI and dashboard. After changing TLS mode, restart the stack service for the change to take effect. The DNS API token (if using ACME DNS-01) is stored in secrets.env — not editable here.

TLS mode Self-signed: browser will show a security warning. Development use only. ACME HTTP-01: automated Let's Encrypt certificate via HTTP challenge (requires port 80 to be publicly reachable). ACME DNS-01: automated Let's Encrypt certificate via DNS challenge (no public port required; requires a supported DNS provider). Manual: provide paths to an existing certificate and private key.
