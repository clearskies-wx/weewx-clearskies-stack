Webcam settings {# ------------------------------------------------------------------ #} {# Enable / disable toggle #} {# ------------------------------------------------------------------ #} When disabled, the webcam card is hidden from the dashboard entirely.
{# ------------------------------------------------------------------ #} {# Config fields — shown only when enabled #} {# ------------------------------------------------------------------ #}
Webcam connection Direct URL to the camera snapshot image (JPEG or PNG). Example: http://192.168.1.100/snapshot.jpg. The dashboard fetches this URL directly from the browser — it must be reachable by visitors. How often the dashboard reloads the snapshot. Minimum 5 s, maximum 3600 s (1 hour). Default: 30 s. Set higher for cameras with slow connections.
Timelapse Absolute path to the directory where snapshot frames are archived for timelapse playback. Leave blank to disable timelapse. Example: /var/lib/weewx/timelapse. Maximum number of frames to keep in the timelapse archive. Older frames are dropped when the limit is reached. Minimum 10, maximum 1000. Default: 288 (one frame every 5 minutes for 24 hours at 30 s interval).
