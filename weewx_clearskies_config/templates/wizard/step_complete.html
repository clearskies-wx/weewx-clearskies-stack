Troubleshooting — manual steps if needed

If the dashboard does not load or shows stale data after a minute or two, you can restart the services manually on your weewx host:

  1. Restart the API to load the new configuration:
    systemctl restart clearskies-api
    Or: clearskies-stack restart api if you are using Docker Compose.
  2. Restart the realtime service to apply realtime.conf:
    systemctl restart clearskies-realtime
