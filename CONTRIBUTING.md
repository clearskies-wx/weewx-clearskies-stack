# Contributing to Clear Skies

Contributions are welcome! This project uses the **Developer Certificate of Origin (DCO)**
to ensure that contributors have the right to submit their work under this project's
[GPL-3.0 license](LICENSE).

## How to contribute

1. Fork the repository and create a feature branch.
2. Make your changes, following the existing code style.
3. Sign off every commit using `git commit -s`. This adds a `Signed-off-by` trailer
   certifying that you wrote the code or have the right to submit it under the project's
   license. See https://developercertificate.org/ for the full DCO text.
4. Open a pull request against `main`.

## Sign-off example

```
git commit -s -m "Add rain rate smoothing to archive poller"
```

This produces:

```
Add rain rate smoothing to archive poller

Signed-off-by: Your Name <your.email@example.com>
```

## Code of conduct

Be respectful and constructive. Weather station operators are a diverse community —
treat every contributor and user with courtesy.
